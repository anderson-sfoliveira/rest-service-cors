## 💻 Sobre o projeto

Projeto criado para estudar sobre CORS.

Projeto desenvolvido seguindo as orientações do guia [**Enabling Cross Origin Requests for a RESTful Web Service**](https://spring.io/guides/gs/rest-service-cors/) oferecido pelo [Spring](https://spring.io/).

---

## 🚀 Como executar o projeto

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina a ferramenta:
[Maven](https://maven.apache.org/).
Além disto é bom ter um editor para trabalhar com o código como [Spring Tools 4 for Eclipse](https://spring.io/tools/).

#### 🎲 Rodando o projeto

```bash

# Clone este repositório
$ git clone https://github.com/anderson-sfoliveira/rest-service-cors.git

# Importe o projeto para dentro do STS4.

# Solicite um "Update Maven Project" (Alt + F5)

# Inicie a aplicação

# O servidor inciará na porta:8080 - acesse http://localhost:8080/greeting ou http://localhost:8080/greeting?name=UserTest

```
